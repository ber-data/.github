# Welcome to BER Data 👋

BER-Data hosts schemas, registries, and supporting tooling for the data lakehouses of the U.S. DOE's Biological and Environmental Research (BER) program.

Schemas are authored in [LinkML](https://linkml.io), a data modeling framework and syntax for describing the structure of data. 

## Primary Resources

- **Documentation:** [ber-data.github.io](https://ber-data.github.io)
- **Lakehouse Registry:** [ber-data.github.io/lakehouse-registry](https://ber-data.github.io/lakehouse-registry/)
- **Modeling language:** [LinkML](https://linkml.io)
- **Program:** [DOE Biological and Environmental Research](https://science.osti.gov/ber)
